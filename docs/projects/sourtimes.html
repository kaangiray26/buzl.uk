The Python Eksisozluk API Wrapper

Ekşi Sözlük has been running for over 23 years now and still, there isn't a proper API for developers.

There isn't much to say, I was bored and coded this piece of cake overnight. You can easily install it via PyPI.

Let's go over some of the features:

  • Get autocomplete results based on a query.
  • List titles from the gündem section.
  • List titles from the debe section.
  • Search for titles with parameters.
  • Show entries from a page.

Sourtimes does not use authorization to access the page and therefore there are some functions we can't make use of, such as getting titles from the bugün section.

Check out the package repo !
