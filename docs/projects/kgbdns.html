Kgb DNS

~

Free self-hosted DDNS service

This project is an implementation of a dynamic DNS service, just like Duck DNS. Actually, I've done this project to get even with them. Here's why:

When I was a kid, I had the best times of my childhood while I was playing Minecraft with my friends in servers. After a while we've built our own and played there with a new map each week. But dealing with IP addresses was a great hassle for us and at that time Duck DNS arrived to help us, without expecting anything from us.

So, this is like a tribute to them, to help others just like Duck DNS helped me. The service uses A records to quickly create subdomains and updates the IP addresses using Gandi's LiveDNS API. Normally, Namecheap is my go-to service when I look for domain names, but Namecheap only offers up to 150 subdomains.

Check the website yourself !
