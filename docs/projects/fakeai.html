Imitating Human Responses

This is such a silly project. A few years ago I was thinking about creating a bot that would imitate other users in Reddit. But, I didn't have the resources to use a neural network model like GPT-3, so I did it my way.

Before you send a message on the website, you select a Subreddit. This will be the source of the data set for the comments. The whole process runs on the client.

Now, when you send a message, the program searches for similar comments on the selected Subreddit and collects replies on those comments. After that it tries to find a suitable reply that can be sent to your message.

The result? Surprisingly funny. And at the same time aggressive, offensive, hideous and disgusting.

Check the website yourself !
