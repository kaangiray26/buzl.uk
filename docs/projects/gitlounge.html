Social Networking off GitHub

OK, don't laugh but this one's too silly. For a while, I had the idea of using parts of GitHub to achieve other functions. This time I tried building a social media platform on top of GitHub.

Because it uses GitHub's API and services, there is no need for a server except when users try to authorize GitHub OAuth. So, I've just used PythonAnywhere to return tokens for users with a simple Flask app.

Now, the social media platform consists of a static website hosted on GitHub as well. The service looks for issues on GitHub with a specific keyword in their titles and displays them as posts.

To show posts in a meaningful way, we had to have a ranking system. Normally, users can leave reactions to issues and there is this one reaction that I've used to sort posts: The thumbs up emoji, üëç

The rest was fairly easy, I've added some functions such as displaying user profiles, searching for posts, creating new posts etc...

Now that I can look back and laugh a little in my room, I can say that this was a fun project and all that it took was just a single day. So, if you see any issues or have some ideas that are possible to integrate, just create a post on GitLounge :)
