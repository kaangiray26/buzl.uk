# Sourtimes
The Python Eksisozluk API Wrapper

![](../project_images/sourtimes.png)

## About
Sourtimes is a Python wrapper for https://eksisozluk.com/

## Features
* Get autocomplete results
* Get ```gündem``` titles
* Get ```debe``` titles
* Search for titles with parameters
* Get entries from a page
