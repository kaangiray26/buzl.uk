# Endless
Unifying the social media experience

![](../project_images/endless.png)

## What is Endless?
Well, nothing. This is just a silly project I made to deal with my boredom. I wanted to create a social media platform but figured out that it would be too empty to gather any attention. So I decided to make a platform that would be empty by design. I wanted to make a platform that would be endless.

## How does it work?
The app uses CapacitorHttp to make client-side requests to the server and avoid CORS issues. This allows us to scrape websites and get their content without having to use a server. What this means for the end-user is that the app is very fast and responsive. The server I am hosting is only used to store upvotes and comments.

## But why?
Endless is similar to those RSS feed readers. However, with all respect to RSS, this allows users to not just read the content but also interact with it. You can upvote and comment on posts. Save posts you find interesting. And even share posts with your friends. All of this without having to create an account or login. Because I hate when a website asks me to create an account just to read an article or like a post. I just want to read the content and move on. And that's what Endless is all about.

Besides, RSS feeds don't allow for pagination. You can only see the latest posts. Endless allows you to endlessly scroll through the content. Hence the name. And also, Endless allows you to see the content on websites that don't have RSS feeds.

## What's next?
At the moment there are only a few websites that are supported. I plan to add more websites in the future but that will take some time. You can suggest websites by creating an issue on GitHub. If you want to help me out, you can also write an extractor for a website and create a pull request. I will be happy to merge it :)

Endless will be free and open-source forever. Enjoy!

## Features
* No account needed
* Save posts to read later
* Upvote and comment on posts
* Remove upvotes and comments if needed
* Fuzzy search on post titles
* Backup and restore your data
