# pitch-perfect
OTP Messaging with a GUI

![](../project_images/pitch-perfect.png)

## About
pitch-perfect is a messaging app that uses One-Time Pad encryption and PGP encryption. It comes with a graphical user interface and the app is written in Python and uses PyQt5 for the GUI.

## Features
* Provides a minimal email client
* Encrypts messages with OTP and PGP
* Uses 1024-byte long keys for OTP
* Uses RSA with size 4096 for PGP
