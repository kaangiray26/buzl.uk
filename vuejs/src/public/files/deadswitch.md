# deadswitch

![](../project_images/deadswitch.png)

## About
> A dead man's switch is a switch that is designed to be activated or deactivated if the human operator becomes incapacitated, such as through death, loss of consciousness, or being bodily removed from control. Originally applied to switches on a vehicle or machine, it has since come to be used to describe other intangible uses, as in computer software. ([Wikipedia](https://en.wikipedia.org/wiki/Dead_man%27s_switch))

Deadswitch is an implementation of a dead man's switch for software. It is a simple application that will send an email to a specified address if the user does not check in within a specified time period. The user can check in by responding to the email with a pre-defined token.
