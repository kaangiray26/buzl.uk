# shortcut
Serverless URL Redirector

![](../project_images/shortcut.png)

## About
shortcut is a URL redirector that allows you to use `github.io` as a domain for your shortcuts. It is completely serverless and uses GitHub Pages to host the website. It is also open-source, so you can host your own instance of shortcut if you want to.

Using shortcut you can spoof URLs to make them harder to guess or bypass domain blocks.
