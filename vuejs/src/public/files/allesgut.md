# Alles gut?
Colloborative Daily Status

![](../project_images/allesgut.png)

## About
Alles gut? is a simple web app that utilizes the power of decentralized databases to store daily status updates. Just show how's your day going by selecting one of the options and it will be stored in the database. The current status will show up on the homepage.
