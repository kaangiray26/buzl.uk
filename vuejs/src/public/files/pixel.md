# Pixel Motion Photos
Convert Pixel Motion Photos to Videos

![](../project_images/pixel.png)

## About
This is a simple tool that converts Pixel Motion Photos to videos using ffmpeg.wasm. So, you can use it on your browser without uploading your photos to a server.

## Features
* Convert photos and display them
* Bulk convert
* Download all videos as a single .zip file
