# FakeAI
Imitating Human Responses

![](../project_images/fakeai.png)

## About
FakeAI uses Reddit's subreddits to find similar comments via Hamming distance and then come up with real responses to these comments. This way, it imitates an AI chatbot while just searching for the best response in a subreddit.
