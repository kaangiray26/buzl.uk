# GitLounge
Social Networking off GitHub

![](../project_images/gitlounge.png)

## About
GitLounge is a hacky social networking platform built on top of GitHub. It uses GitHub's APIs to submit posts and comments. Use your GitHub account to login and start posting. You can also see trending and hot posts on the homepage.
