# gitlinker
Serverless URL Shortener

![](../project_images/gitlinker.png)

## About
gitlinker is an idiomatic URL shortener that uses GitHub Team Discussions as a database. The entered links are saved to discussions via GitHub API and then the links are retrieved from the same endpoint and redirected from the website.
