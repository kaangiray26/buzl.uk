# Sync Everything
A Firefox Extension to synchronize video playback between P2P devices

![](../project_images/sync-everything.png)

## About
Sync allows you to synchronize your videos with your friends without using any servers. You can use it on nearly all websites by utilizing the HTML5 video element. Connections are P2P and there are no data being collected or sent over the horizon.

## Features
* Play/pause actions
* Seek action
* Works on all websites that use HTML5 video element (local files included)!
