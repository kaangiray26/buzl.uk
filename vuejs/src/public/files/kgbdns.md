# KgbDNS
Free self-hosted DDNS service

![](../project_images/kgbdns.png)

## About
KgbDNS is a free self-hosted DDNS service. It allows you to create a subdomain for your home network and update it with your public IP address. It is built with Python and Flask. You can also use the source code to host your own DDNS service.

Check it out here:
* [GitHub](https://github.com/kaangiray26/kgbdns-pub)
